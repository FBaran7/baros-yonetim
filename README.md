# baros-yonetim 🛍️

Inventory, profit/loss, and product tracking application built for BAROS Menswear's 
Trendyol marketplace operations.

## 🎯 Purpose

A practical operations tool I built to manage the day-to-day needs of running an active 
e-commerce store: tracking stock levels, monitoring profit margins, and managing the 
product catalog.

## ✨ Features

- 📦 Stock tracking and inventory management

- 💰 Profit and loss calculation

- 🏷️ Product catalog management

- 📊 Operational visibility for marketplace decisions

## 🧠 Development Approach

Built using AI-assisted development (ChatGPT) following a step-by-step iterative approach. 
The product requirements come directly from real BAROS operational needs, since I run 
the marketplace operations myself.
