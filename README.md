# baros-yonetim

Inventory, profit/loss, and product tracking application built for BAROS Menswear's Trendyol marketplace operations.

## Overview

A practical operations tool I built to manage the day-to-day needs of running an active e-commerce store on Trendyol: tracking stock levels, monitoring profit margins, and managing the product catalog. Product requirements come directly from real operational needs, since I run BAROS's marketplace operations myself.

## Features

- Stock tracking and inventory management

- Profit and loss calculation

- Product catalog management

- Operational visibility for marketplace decisions

## Development Approach

Built using AI-assisted development (ChatGPT) following a step-by-step iterative approach. Each module was scoped, implemented, and tested incrementally based on real operational needs from BAROS's Trendyol store.
